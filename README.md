# HR Analytics Dashboard (Power BI)

A Power BI report (HR.pbix) providing HR analytics and workforce insights.

## Report Pages

- Home – Overview / landing page summarizing key HR metrics
- Detail – Detailed breakdown of HR data
- Action – Actionable insights / drill-through page

## Getting Started

1. Download HR.pbix
2. Open it in Power BI Desktop (free): https://powerbi.microsoft.com/desktop/
3. If the report connects to a live data source, you may need to update the data source credentials under Transform Data > Data Source Settings

## Requirements

- Power BI Desktop (Windows) or a Power BI Pro/Premium license to view via Power BI Service
